<메모리> - 메모리에는 정수값 하나만 들어갈 수 있음
그그그거 -> 메모리 3번, 메모리 3번의 값
그그그 -> 숫자 3
그그그거거 -> (메모리 3번에 저장된 숫자)번 메모리

<함수>
뭐지 -> 입력값을 메모리에 저장 ( 메모리 뭐지)
뭐냐 -> 출력 (값 뭐냐)
뭐더라 -> 메모리에 값을 할당 (메모리 뭐더라 값)
있잖아 -> n번 아래 줄로 이동 (n이 음수면 위로감) (값 있잖아)
진짜뭐지 -> 한 글자를 입력받아 유니코드(10진법)를 저장
진짜뭐냐 -> 값에 해당하는 유니코드를 가지고 있는 글자를 출력

<연산> - 모든 연산은 중위표기법이며, 연산자 우선순위가 적용됨
, -> 덧셈
,, -> 뺄셈
. -> 곱셈
.. -> 몫
... ->나머지
아 어 -> 괄호

<비교> - 참이면 1, 거짓이면 0의 값을 가짐
~ -> 같음
; -> 큼
;; -> 크거나 같음
